# KY School Absenteeism Dataset (2024-2025) 

## Project overview
The goal for this project is to see if there is a correlation between absenteeism and dropout rate. I've decided first to look at the absenteeism data then I will continue and look at the dropout rate.

## Key findings
Jefferson County has the highest number of absenteeism totals in this dataset.
Most counties appear to have fairly low number of absenteeism totals.
Male Absenteeism counts are slightly higher Female Absenteeism counts

I'll need to look at historical data and enrollment data for a more holistic view of what is going on with this data, before I jump to conclusions...

## Known issues
Just because counts may be higher for one group over another doesn't necessarily mean this is an accurate depiction of what the data is telling us. Some counties will have higher enrollment data than others. Jefferson County is a great example of this. It's a heavily populated county and this can make it appear that it has higher % of absenteeism than other counties.
